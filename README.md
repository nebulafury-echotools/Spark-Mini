# Spark Mini
Mobile version of the Spark

**IF YOU ARE ON WINDOWS, USE THE MAIN SPARK INSTEAD: https://github.com/heisthecat31/Spark**

Application for non-Windows clients. Won't be able to have a lot of the features of the main Spark, but hopefully still can be useful to people.
Windows clients can run EchoVR itself, allowing for a larger feature set.

## Installing on iOS with AltStore

Spark Mini can be sideloaded on iPhone or iPad with AltStore using the `.ipa` release file.

### Requirements

- An iPhone or iPad
- A Mac or Windows PC
- AltStore installed on your iOS device
- AltServer running on your computer
- The latest Spark Mini `.ipa` file from this repository's releases

### Steps

1. Install AltStore by following the official AltStore setup guide: https://altstore.io
2. Download the latest Spark Mini `.ipa` file from the Releases page.
3. Make sure your iPhone or iPad is connected to the same Wi-Fi network as the computer running AltServer.
4. Open the AltStore app on your iOS device.
5. Go to the **My Apps** tab.
6. Tap the **+** button in the top-left corner.
7. Select the Spark Mini `.ipa` file you downloaded.
8. Wait for AltStore to finish installing the app.

If iOS blocks the app from opening, go to **Settings > General > VPN & Device Management**, select your Apple ID, and trust the developer profile.

AltStore apps installed with a free Apple ID need to be refreshed at least once every 7 days. Keep AltServer running on your computer and open AltStore's **My Apps** tab to refresh Spark Mini before it expires.
